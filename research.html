Projects

AI-Based Crop Disease Detection System

Developed a CNN-based system for early identification of crop diseases using image processing techniques. The project emphasized model training and hardware-aware AI deployment strategies.

Design and Analysis of CMOS Low Noise Amplifier

Designed and analyzed a CMOS Low Noise Amplifier using source inductive degeneration to improve gain and noise performance. Simulations were carried out using Cadence Virtuoso.

Smart Street Lighting System

Implemented an IoT-based smart street lighting system using Arduino for energy-efficient and automated lighting control.
